گزارش پروژه درس یادگیری ماشین

طراحی و پیاده‌سازی اپلیکیشن استعدادیابی ورزشی در رشته تنیس با استفاده از یادگیری ماشین و تشخیص وضعیت بدن (Pose Detection)

مقدمه (Introduction)
استعدادیابی ورزشی یکی از مهم‌ترین چالش‌ها در دنیای ورزش حرفه‌ای است. شناسایی زودهنگام استعدادها می‌تواند نقش تعیین‌کننده‌ای در موفقیت ورزشکاران و بهینه‌سازی منابع آموزشی و تمرینی داشته باشد. با پیشرفت‌های اخیر در حوزه یادگیری ماشین (Machine Learning) و بینایی ماشین (Computer Vision)، امکان تحلیل داده‌های انسانی به‌صورت دقیق‌تر و علمی‌تر فراهم شده است.
در این پروژه، یک اپلیکیشن هوشمند استعدادیابی برای ورزش تنیس طراحی و پیاده‌سازی شده است که با استفاده از:
داده‌های متنی (ویژگی‌های فیزیکی و فردی)
داده‌های ویدئویی (حرکات بدن ورزشکار)
قادر است استعداد فرد را برای ورزش تنیس ارزیابی کرده و نتیجه‌ای کمی و قابل تفسیر ارائه دهد. این سیستم ترکیبی از یادگیری ماشین کلاسیک و تحلیل بیومکانیکی حرکات بدن است و نمونه‌ای از یک سیستم Multi-modal AI محسوب می‌شود.

هدف پروژه (Project Objectives)
اهداف اصلی این پروژه عبارت‌اند از:
طراحی یک سیستم هوشمند برای استعدادیابی ورزشی در رشته تنیس
استفاده همزمان از داده‌های عددی و ویدئویی
استخراج ویژگی‌های بیومکانیکی مؤثر در عملکرد تنیس‌باز
آموزش یک مدل یادگیری ماشین برای پیش‌بینی استعداد
پیاده‌سازی یک رابط کاربری ساده و قابل استفاده برای کاربران
این پروژه با تمرکز بر کاربرد عملی یادگیری ماشین در دنیای واقعی طراحی شده و قابلیت توسعه برای سایر رشته‌های ورزشی را نیز دارد.

معماری کلی سیستم (System Architecture)
ساختار کلی سیستم به صورت زیر طراحی شده است:
ورودی کاربر
اطلاعات فردی (سن، قد، وزن)
ویژگی‌های فیزیکی (سرعت، قدرت، انعطاف‌پذیری)
ویدئوی حرکت کاربر (ضربه تنیس)
پردازش داده
پردازش داده‌های متنی با روش‌های یادگیری ماشین
پردازش ویدئو با الگوریتم Pose Detection
مدل یادگیری ماشین
استخراج ویژگی
آموزش مدل
پیش‌بینی استعداد
خروجی
نمایش نتیجه استعدادیابی
احتمال موفقیت در تنیس
این معماری باعث جداسازی مناسب بخش‌های مختلف سیستم و افزایش خوانایی و توسعه‌پذیری پروژه شده است.

داده‌ها و ویژگی‌ها (Dataset & Features)
4.1 داده‌های متنی
ویژگی‌های متنی مورد استفاده در این پروژه شامل موارد زیر هستند:
سن (Age)
قد (Height)
وزن (Weight)
انعطاف‌پذیری (Flexibility)
سرعت (Speed)
قدرت بدنی (Strength)
این ویژگی‌ها به‌صورت عددی و نرمال‌شده به مدل داده می‌شوند.
4.2 داده‌های ویدئویی
برای تحلیل حرکات بدن، از ویدئوی اجرای ضربه تنیس استفاده می‌شود. با کمک الگوریتم Pose Detection، نقاط کلیدی بدن استخراج شده و زوایای مهم محاسبه می‌شوند.
ویژگی‌های بیومکانیکی استخراج‌شده:
زاویه آرنج (Elbow Angle)
زاویه زانو (Knee Angle)
این زوایا نقش مهمی در کیفیت ضربه، انتقال قدرت و تعادل بدن در تنیس دارند.

تشخیص وضعیت بدن (Pose Detection)
برای تشخیص وضعیت بدن از کتابخانه MediaPipe استفاده شده است. این ابزار با دقت بالا قادر به شناسایی نقاط کلیدی بدن انسان در هر فریم ویدئو می‌باشد.
مراحل انجام Pose Detection:
دریافت فریم‌های ویدئو
شناسایی نقاط کلیدی بدن
استخراج مختصات مفاصل
محاسبه زوایای مفصلی با روابط هندسی
زاویه هر مفصل با استفاده از ضرب داخلی بردارها و محاسبه زاویه بین آن‌ها به دست می‌آید. میانگین زوایا در کل ویدئو به‌عنوان ویژگی نهایی استفاده می‌شود.

مدل یادگیری ماشین (Machine Learning Model)
برای انجام استعدادیابی از الگوریتم Random Forest Classifier استفاده شده است. دلایل انتخاب این مدل عبارت‌اند از:
عملکرد مناسب روی داده‌های کوچک
مقاومت در برابر overfitting
قابلیت تفسیرپذیری بالا
دقت مناسب در مسائل طبقه‌بندی
6.1 برچسب‌گذاری داده‌ها
خروجی مدل به صورت یک مسئله طبقه‌بندی دودویی تعریف شده است:
1 : استعداد مناسب برای تنیس
0 : استعداد نامناسب برای تنیس
6.2 آموزش مدل
داده‌ها به دو بخش آموزش و آزمون تقسیم شده و مدل روی داده‌های آموزشی آموزش داده می‌شود. پس از آموزش، مدل ذخیره شده و در اپلیکیشن مورد استفاده قرار می‌گیرد.

پیاده‌سازی اپلیکیشن (Application Implementation)
برای پیاده‌سازی رابط کاربری از Streamlit استفاده شده است. این ابزار امکان ساخت سریع اپلیکیشن‌های یادگیری ماشین را فراهم می‌کند.
قابلیت‌های اپلیکیشن:
دریافت اطلاعات کاربر از طریق اسلایدرها
آپلود ویدئوی حرکت تنیس
تحلیل خودکار ویدئو
پیش‌بینی استعداد
نمایش نتیجه به‌صورت گرافیکی و متنی
رابط کاربری ساده و کاربرپسند طراحی شده تا حتی کاربران غیرمتخصص نیز بتوانند به راحتی از سیستم استفاده کنند.

نتایج و تحلیل (Results & Analysis)
خروجی سیستم شامل موارد زیر است:
پیش‌بینی استعداد تنیس
احتمال موفقیت (Confidence Score)
نمایش زوایای مفصلی استخراج‌شده

نتیجه‌گیری و پیشنهادات آینده (Conclusion & Future Work)
در این پروژه یک سیستم هوشمند برای استعدادیابی تنیس با استفاده از یادگیری ماشین و تحلیل ویدئویی طراحی و پیاده‌سازی شد. نتایج نشان داد که ترکیب داده‌های متنی و بیومکانیکی می‌تواند دقت تصمیم‌گیری را افزایش دهد.
پیشنهادات برای توسعه آینده:
افزایش حجم دیتاست
استفاده از مدل‌های عمیق (Deep Learning)
تحلیل حرکات پیشرفته‌تر مانند Forehand و Backhand
توسعه اپلیکیشن
گسترش سیستم به سایر رشته‌های ورزشی
